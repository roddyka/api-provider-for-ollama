Se ouviu sobre as últimas atualizações do WordPress já sabe! 🚀

O 7.0 veio com integração nativa a AI — para facilitar uma série de coisas como ajustes em textos, geração de conteúdo e muito mais directamente no editor do WordPress.

Notei um pequeno problema: não havia conector para Ollama, que é o que uso localmente.

Dito isto, um desafio criado! 💡

Desenvolvi um plugin WordPress open-source que liga o novo AI Client do WordPress 7.0 ao Ollama — o runtime de AI local que permite correr modelos como Llama 3, Mistral, Gemma ou Phi no teu próprio servidor ou máquina local.

**O que o plugin faz:**

✅ Zero API key — sem subscrições, sem cloud, sem custos por token
✅ Descobre automaticamente os modelos instalados no teu Ollama
✅ Aparece directamente em Definições → Conectores no wp-admin
✅ Funciona em localhost ou servidor remoto
✅ 100% GPL — free e open-source

Ideal para quem quer experimentar as funcionalidades de AI do WordPress sem ter de assinar uma API key, ou para quem prefere manter os dados em casa.

Vou partilhar no repositório para que todos os interessados em usar AI no WordPress — mesmo que a teste local ou num servidor com o Ollama a correr — possam fazê-lo sem depender de nenhum serviço externo.
